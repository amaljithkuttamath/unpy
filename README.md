# unpy

# Steps to get started:

1. Clone this repository.
2. Open terminal in the directory.
3. Commands to use:
    
    Usage: [./unpy check 3 del ] [ ./unpy check 2 del ] [ ./unpy check all ] [ ./unpy check 3 ] [ unpy check 2 ]
 

Details of each command.
